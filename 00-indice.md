# Centelhas Divinas — Índice do Worldbuilding

Estrutura para organizar todo o material de worldbuilding antes de escrever os capítulos com o Claude como ghostwriter.

---

## Estrutura de pastas

| Pasta | O que colocar aqui |
|-------|--------------------|
| **01-facoes** | Facções, alianças, governos (ex: Eixo Vermelho, Aliança Livre) |
| **02-personagens** | Personagens principais, secundários, antagonistas |
| **03-mundo-e-cenario** | Geografia, tecnologia, regras do mundo, atmosfera |
| **04-timeline-e-historia** | Linha do tempo, eventos históricos, como chegamos aqui |
| **05-plot-e-arqueargos** | Arcos narrativos, conflitos centrais, temas |
| **06-capitulos** | Rascunhos e capítulos escritos (para o ghostwriting) |

---

## Como usar

1. **Preencha cada pasta** com notas sobre o seu mundo.
2. **Use links** entre notas (ex: `[[Eixo Vermelho]]` no Obsidian) para conectar ideias.
3. **Quando estiver pronto para escrever**, peça ao Claude para gerar capítulos usando todo esse contexto.
4. O Claude segue automaticamente as [[diretrizes_de_escrita|Diretrizes de Escrita]] (Bíblia de Estilo) ao escrever.
